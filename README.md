# Developer Sai — Cyberpunk Portfolio

A premium, futuristic cyberpunk full-stack developer portfolio website.

## Quick Start

Simply open `index.html` in any modern browser. No build step required.

## Deploy to GitHub Pages

1. Create a new GitHub repository
2. Upload all files from this folder to the repository root
3. Go to **Settings → Pages**
4. Set source to **main branch / root**
5. Your site will be live at `https://yourusername.github.io/repo-name`

## Deploy to Netlify

Drag and drop the entire `developer-sai-portfolio/` folder onto [app.netlify.com/drop](https://app.netlify.com/drop).

## Deploy to Vercel

```bash
npm i -g vercel
vercel --prod
```

## Folder Structure

## Features

- Dark Cyberpunk theme with 6 switchable themes
- Glassmorphism cards
- Animated particle canvas background
- Mouse glow + parallax effects
- Scroll reveal animations
- Animated stat counters
- Auto-sliding testimonials carousel
- Pricing / Maintenance monthly/yearly toggle
- AI Chatbot widget
- Project request form with WhatsApp redirect
- Contact form
- Floating WhatsApp + Back-to-Top buttons
- Fully responsive (mobile, tablet, desktop)
